# 코딩 테스트 프로젝트

## 📝 프로젝트 설명



## 🚀 설치 및 실행

1.  **의존성 설치**

    ```bash
    npm install
    ```

2.  **프로그램 실행**

    (실행할 메인 파일이 `src/index.js`라고 가정)
    ```bash
    node src/index.js
    ```
    또는 `package.json`에 스크립트를 등록하여 실행할 수 있습니다.

    ```json
    // package.json
    "scripts": {
      "start": "node src/index.js"
    }
    ```

    ```bash
    npm start
    ```

## ✅ 테스트

Jest를 사용하여 테스트를 실행합니다.

```bash
npm test
```
